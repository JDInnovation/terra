Batalha de Terras

Turno do Jogador 1

Energia Restante: 5

Ação: Coloque sua base inicial

Instruções

O objetivo do jogo é destruir a base do adversário. Cada jogador tem 5 pontos de energia por turno para realizar ações.

As ações disponíveis são:

  • Colocar Base: Coloca uma base inicial.
  • Adicionar Personagens: Adiciona guerreiros, arqueiros, tanques ou healers ao tabuleiro.
  • Mover Personagem: Mova seu personagem para uma célula adjacente.
  • Atacar: Ataca um personagem inimigo adjacente.
  • Usar Habilidade Especial: Habilidade única que pode ser utilizada uma vez por jogo.
